About

This game uses a fire extinguisher robot characters. The robot will explore the maze-shaped room to put out the fire that burns in the room. Object form of the cat should be saved. If the robot crashed into the wall, the robot will go back to the first state the circumstances for each level.

if it can not extinguish the fire, the game ends. However, if the player managed to extinguish the fire and save the cat it will switch difficulty level at the higher level.
